# Portfolio Website

This is a portfolio website built using HTML, CSS, and JavaScript.

The website showcases my profile and a small selection of demonstration projects related to programming and web development. It was created as part of a technical assessment and to present my work in a clear and structured way.

## Features
- Simple navigation
- Profile section
- Projects section
- About section
- Basic JavaScript interaction (Back to Top button)

## Technologies Used

- CSS
- JavaScript
